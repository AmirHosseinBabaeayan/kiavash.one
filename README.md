# kiavash.one

This repo uses Mira

* documents: [Mira](https://miraxy.github.io/)
* github: [Mira](https://github.com/kiamazi/mira)

## install mira

``` bash
sudo apt install git make build-essential
sudo cpan -i -T Mira
```

## Building

``` bash
git clone https://github.com/kiamazi/kiavash.one.git
cd kiavash.one
mira build
mira view
```
